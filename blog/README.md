# /skills.md: The New Interface for AI Agents

This directory contains a comprehensive blog post about the `/skills.md` pattern for agent-website interaction, using augmi.world/skills as a case study.

## Contents

- **`the-skills-md-pattern.md`** - The full blog post in Markdown format
- **`blog-hero-image.svg`** - Hero image for the blog post
- **`blog-diagram-interaction.svg`** - Diagram showing agent -> skills.md -> website interaction flow
- **`blog-augmi-mockup.svg`** - Mockup of augmi.world/skills interface

## Summary

The `/skills.md` pattern is an emerging convention where websites expose a machine-readable manifest of their capabilities at a predictable URL endpoint. This blog post explores:

- The problem of agents without context
- How the `/skills.md` pattern solves it
- Case study: augmi.world/skills implementation
- The Agent Skills standard from agentskills.io
- How agents discover and use skills
- The ecosystem (ClawHub, Claude Code, etc.)
- Future implications for the agentic web

## Key Insights

1. **Discoverability**: Agents can find capabilities automatically
2. **Interoperability**: Write once, use everywhere
3. **Extensibility**: Add capabilities without code changes
